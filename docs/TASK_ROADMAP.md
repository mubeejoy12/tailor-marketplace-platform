# Development Roadmap

This document defines the order features must be built.

---

# Phase 1

Project setup.

Tasks

Create Spring Boot backend.

Create React frontend.

Connect backend to database.

---

# Phase 2

Authentication.

Tasks

Create User entity.

Create login and registration APIs.

Implement JWT authentication.

---

# Phase 3

Tailor profiles.

Tasks

Create Tailor entity.

Allow tailors to register.

Admin approves tailor accounts.

---

# Phase 4

Design catalog.

Tasks

Tailors upload clothing designs.

Customers browse designs.

---

# Phase 5

Measurement system.

Tasks

Customers save body measurements.

Measurements used during order.

---

# Phase 6

Order system.

Tasks

Customers place orders.

Tailors receive orders.

Order status updates.

---

# Phase 7

Review system.

Tasks

Customers rate tailors.

Average rating displayed.

---

# Phase 8

Deployment.

Tasks

Deploy backend.

Deploy frontend.

Connect production database.
